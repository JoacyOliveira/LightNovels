## Instalação

### Clone o projeto
```
git clone https://github.com/JoacyOliveira/LightNovels.git
cd LightNovels 
```

### virtualenv
```
python -m venv venv
```
Windows
```
venv\Scripts\activate
```



## Instale as dependencias
```
pip install -r requirements.txt
python manage.py runserver
```
